BERSERK, manga creado por Kentaro Miura, es un seinen no apto para todo público. Todo aquel que lo haya leído afirmará que es distinto al resto de mangas.

Un hombre marcado por seres de otro mundo, una venganza y batallas cruentas en el que el protagonista hace gala de sus habilidades para blandir una espada gigantesca. Humanos, demonios y seres de otro mundo peleando entre ellos para dominar al otro. Esta es la historia de Berserk.
