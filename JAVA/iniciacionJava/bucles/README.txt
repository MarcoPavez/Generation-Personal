Se crea HASHMAP con ingreso int en key y arraylist en valores

se usa bucle while según número de estudiantes a tratar

dentro del while se hace un for para ingresar notas del estudiante (limitado a 3)
el final de este for se agrega la nota ingresada al array de notas

dentro del for se hace un while que limita el rango de notas ingresadas entre 1 a 7

fuera de todo bucle se ingresa al hashmap los nombres de alumno y sus notas
además, fuera del bucle se resta valor a cantidad de alumnos ingresada para llevarlo a 0 y terminar el while principal
